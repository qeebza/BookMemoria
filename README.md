# BookMemoria

BookMemoria is a book tracking web application built from scratch using PHP.

## Goals

- Learn backend engineering
- Build my own MVC framework
- Learn routing
- Learn authentication
- Learn REST principles
- Deploy to a real server

## Tech Stack

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Test Run
